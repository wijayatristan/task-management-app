# Task Management App

Simple full-stack task management application built for the Front-End & Back-End Developer Internship Technical Test.

## Tech Stack

### Frontend
- Next.js (App Router)
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- SQLAlchemy
- PostgreSQL
- JWT Authentication

## Project Status

🚧 Planning Phase

This repository is currently in the planning stage. Development will be completed in multiple phases, starting from requirement analysis, system design, backend implementation, frontend implementation, integration, testing, and documentation.
